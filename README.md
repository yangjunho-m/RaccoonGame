# RaccoonGame
Unity 엔진을 사용하여 라쿤키우기 게임을 제작했다.  <br>
시작 2021-1-21 <br>

에셋 구매 없이 하나하나 그림 그리고 디자인하였습니다. <br>

클릭하면 코인을 얻고 라쿤 지능을 높여 코인 획등량을 높일 수 있습니다. <br>
코인을 모아서 사람, 먹이, 보물 등을 구매하여 코인을 더욱 많이 벌 수 있습니다. <br>
구글 로그인 기능을 구현하고 구글 콘솔에 게임을 등록하여 <br>
플레이스토어에 출시하고 친구들과 오류를 찾으며 오류를 고쳤습니다. <br><br>

<img width="349" height="534" alt="image" src="https://github.com/user-attachments/assets/dc7feee9-02bb-4267-8aff-d71bd7fb4397" /> <br>

현재는 군대 갔다 온 이후 정책 위반 (계정 사용안함) 으로 개발자 계정이 해지되어 플레이 할 수 없습니다.
